## 1. 매크로 환경 종합 평가

### 1.1 한국은행 기준금리

| 시점 | 기준금리 | 변화 |
|------|---------|------|
| 2025-01 | 1.50% | — |
| 2025-02~03 | 1.25% | ↓ 25bp 인하 |
| 2025-04 | 1.25% | 동결 |
| 2025-05~12 | 1.00% | ↓ 25bp 추가 인하 |
| 2026-01~현재 | 1.00% (MACRO-DATA 기준) | 동결 |

> **참고**: 외부 소스(Trading Economics, BOK 공식)에 따르면 2026년 2월 BOK 기준금리는 **2.50%**로 표시됩니다. MACRO-DATA 파이프라인의 값(1.00%)과 차이가 있어 **데이터 검증이 필요**합니다. 본 분석에서는 **외부 소스 2.50%를 실제 기준금리로 채택**하되, 통계 모델 결과는 MACRO-DATA 값 기준입니다.

**방향성 판단**: 2024년 10월부터 누적 100bp 인하 이후 현재 **동결 국면**. BOK는 추가 인하 언급을 삭제하고 지정학적 리스크와 자본 유출 압력에 대한 경계를 강조.

**확신도: 75%** — BOK는 2026년 상반기 동결 유지 후, 하반기 25bp 인하 여부는 GDP 성장률(1.8% 전망)과 원/달러 환율에 의존.

### 1.2 물가 동향

| 지표 | 최근값 | 추세 |
|------|--------|------|
| CPI YoY | 변동 (데이터 스케일 이슈) | — |
| PPI YoY | **4.0%** (Dec 2025) | ↓ 6.0%(Sep) → 4.0%(Dec) 하락 |
| 인플레이션 기대 | 안정적 | BOK 전망 2.1% |

**해석**: PPI YoY가 6% → 4%로 하향 안정화 중. 이는 산업재 원가 부담 완화를 시사.
다만 PPI_YOY는 **MACRO-DATA 내 가장 유의미한 한국 매크로 변수** (t-stat 최대 3.22, 80개 이벤트-섹터 조합 중 1위).

**확신도: 70%** — PPI 하향 안정은 제조업 마진 개선에 긍정적. 단, 원화 약세 시 수입물가 재상승 리스크.

### 1.6 매크로 환경 종합 요약

```
금리:   동결 (완화 사이클 종료, 추가 인하 여력 제한적)
물가:   PPI 하향 안정, 소비자물가 2%대 안착
고용:   변동성 높으나 구조적 견고
생산:   12월 121.1 — 연중 최고, 제조업 호조
사이클: Expansion 8개월째 → 후기 확장 진입
```

**종합 판단**: 한국 경제는 **"완만한 확장 + 제한적 통화 완화"** 국면. 달러 약세(DXY 121→118)와 맞물려 원화 강세 압력이 수출주에는 부담이나, 내수/금융주에는 순풍. 반도체 슈퍼사이클이 IP 상승을 견인하는 구조.

## 3. 거시지표-섹터 통계적 영향 분석

> **v2 경고**: 본 섹션의 회귀분석 결과는 구조적 결함이 있습니다.
> (1) expected_value가 전무하여 서프라이즈 계산 불가 — 원시 수준값(level)으로 회귀,
> (2) CPI/CCSI/Employment/INFL_EXPECT 데이터가 손상됨,
> (3) Employment와 INFL_EXPECT가 동일 데이터(r=1.000).
> 상세: [부록 A](#부록-a-투자-가설-오류-분석) 참조.

### 3.1 OLS 회귀분석 개요

| 항목 | 수치 |
|------|------|
| 총 이벤트 | 15개 |
| 총 섹터 | 11개 (+ 2 인덱스) |
| 총 테스트 | 1,170건 (이벤트 × 섹터 × 시간윈도우) |
| FDR α=0.05 유의 결과 | **0건 (0%)** |
| p < 0.01 결과 (보정 전) | **3건** |

> **핵심 한계**: 한국 데이터는 FDR 보정(Benjamini-Hochberg) 후 유의한 결과가 **0건**. 이는 미국(17.8%, 565건)과 극명한 차이. 원인: ① 한국 데이터 기간이 짧음(2023.03~, 33~34 관측치), ② 이벤트 서프라이즈 변동이 작음, ③ 한국 시장의 외부 변수(미국 증시, 원/달러) 의존도가 높아 자국 매크로 지표의 설명력이 낮음.

### 3.3 이벤트별 핵심 발견

#### PPI YoY — 가장 강한 설명력

```
PPI YoY 상승(서프라이즈) 시:
  Healthcare  → win5  +2.31% (t=3.22) ★★★
  Industrials → win10 +1.60% (t=2.02) ★★
  Industrials → win5  +1.40% (t=1.52) ★
  Comm. Svc   → win5  +1.18% (t=1.54) ★
  Materials   → win5  +1.74% (t=1.02)
  Utilities   → win10 -2.87% (t=-1.46) [역방향]
```

**해석**: PPI 상승 서프라이즈는 Healthcare와 Industrials에 양(+)의 영향. 생산자물가 상승 → 가격 전가 여력이 큰 헬스케어/산업재가 수혜. 유틸리티는 비용 부담 증가로 역(-)반응.

#### BOK 기준금리 — 가장 넓은 섹터 영향

```
BOK 금리 인상(서프라이즈) 시:
  IT          → win21 +3.96% (t=1.84) ★★   [반직관적]
  Comm. Svc   → win21 -2.37% (t=-2.89) ★★★
  Energy      → win21 -5.39% (t=-1.83) ★★
  Utilities   → win5  +0.85% (t=1.82) ★★
  Cons. Disc  → win21 +1.33% (t=1.52) ★
  Industrials → win5  -0.59% (t=-1.65) ★
```

**해석**: 금리 인상 시 IT와 유틸리티가 긍정 반응(반직관적). 가능한 설명: ① 금리 인상 = 경제 정상화 신호 → 성장주 재평가, ② 한국 IT 섹터는 금리보다 글로벌 AI 수요에 더 민감. Communication Services와 Energy는 금리 부담으로 음(-)반응.

#### 실업률 — 소비 섹터 양극화

```
실업률 상승 시:
  Cons. Disc   → win21 +0.60% (t=3.04) ★★★ [역방향]
  Cons. Staples → win5  -0.36% (t=-2.70) ★★★
  Cons. Staples → win10 -0.23% (t=-2.57) ★★
  Healthcare    → win5  -0.52% (t=-1.58) ★
```

**해석**: 실업률 상승 시 필수 소비재(Staples)가 하락하고 임의 소비재(Discretionary)가 상승하는 역설적 패턴. 가설: 실업률 상승 → 정부 재정 부양 기대 → 소비 관련 정책 수혜 기대감 반영.

### 전제 2: "서프라이즈(actual - expected) 기반 회귀분석이다"

**결론: 완전히 틀렸다.**

```
expected_value 열이 채워진 한국 이벤트: 0건 / 542건 (0%)
```

이벤트 서프라이즈 모델의 핵심 공식:

```
정상적 모델:  Sector_Return = α + β × (Actual - Expected) + ε
실제 모델:    Sector_Return = α + β × Actual_Level + ε    ← 서프라이즈가 아님
```

**왜 이게 문제인가**:
- CPI가 75,892에서 72,068으로 바뀌면, 모델은 이것을 "서프라이즈"로 해석
- 실제로 이 두 값은 모두 의미 없는 손상 데이터
- 원시 수준값(level)과 섹터 수익률 간의 상관관계는 **허구적 상관(spurious correlation)**
- 이것이 FDR 보정 후 유의 결과가 0건인 근본 원인

### 전제 3: "1,170건의 OLS 테스트는 의미 있는 샘플이다"

**결론: 숫자에 속았다.**

| 항목 | 겉보기 | 실체 |
|------|--------|------|
| 총 테스트 | 1,170건 | — |
| 유효 이벤트 (데이터 있음) | 8개 | 7개 중복 제거 시 (EMPLOYMENT=INFL_EXPECT) |
| 이벤트당 관측치 | 33~34개 | 3년치 월별 데이터. 통계적 검정력(power) 심각 부족 |
| 시간 윈도우 | 6개 | win1/win3/win5/win10/win21 — 다중비교(multiple comparison) 문제 악화 |
| FDR 유의 결과 | **0건** | 어떤 이벤트-섹터 조합도 5% 수준에서 유의하지 않음 |

**핵심**: 1,170건이라는 숫자는 "7개 이벤트 × 11개 섹터 × 6개 윈도우"를 곱한 것일 뿐, 독립적 정보가 1,170개인 것이 아니다. 실질적으로는 **33개 관측치로 462개의 가설을 동시에 테스트**하는 극단적 과적합 상황.

## A.2 예상 못한 변수 (Unexpected Variables)

### 변수 1: 데이터 파이프라인 자체가 결함

초판 분석 시점에서 **"데이터가 수집되어 있다 = 데이터가 정확하다"**라고 가정했다.
이것은 가장 위험한 전제였다.

**발견된 파이프라인 버그 목록**:

| 버그 | 원인 추정 | 영향 |
|------|----------|------|
| CPI_YOY 값이 수천~수십만 | BOK API `902Y014` 시리즈에서 원계열(raw level)을 가져옴. YoY% 변환 로직 누락 | CPI 관련 모든 분석 무효 |

| POLICY_RATE 1.5%p 괴리 | BOK `722Y001` 시리즈에서 잘못된 항목 코드 사용 (금통위 결정금리 vs. 기타 금리) | 금리 서프라이즈 방향/크기 왜곡 |
| 6개 지표 전체 NULL | API 연결 미완성 또는 파라미터 미설정 | 전체 분석 범위 40% 축소 |

### 변수 2: 한국 시장의 미국 종속성

한국 매크로가 아무리 정확해도, **KOSPI의 단기 변동성은 80%+ 미국 요인**(S&P500, FOMC, CPI)에 의해 결정된다.

초판에서 이 사실을 언급만 하고 모델에 반영하지 않았다:
- 미국 CPI 발표일에 KOSPI가 ±2% 움직이는 반면
- 한국 CPI 발표일에 KOSPI는 ±0.3%도 안 움직임
- OLS 모델에 미국 매크로 변수가 없으므로 **누락 변수 편향(omitted variable bias)**

### 변수 3: 이벤트 날짜의 비독립성

한국 매크로 이벤트들은 대부분 **같은 달 초에 발표**된다. Win21 히트맵에서 모든 이벤트가 동일한 섹터 수익률을 보인 것은 이 때문. 미국은 CPI(10~13일), NFP(첫째 금요일), FOMC(6주마다) 등 발표일이 분산되어 있어 이벤트 간 독립성이 높지만, 한국은 그렇지 않다.

## A.3 데이터 해석 오류 (Interpretation Errors)

### 오류 1: "FDR 유의 결과 0건"을 경고로만 처리하고 분석을 계속 진행

**초판의 태도**:
> "핵심 한계: 한국 데이터는 FDR 보정 후 유의한 결과가 0건... 원인: 데이터 기간이 짧음"

이것은 **원인 진단이 틀렸다**. 진짜 원인:
1. 데이터 자체가 손상되어 서프라이즈 계산이 불가
2. 서프라이즈 없이 수준값(level)으로 회귀하면 노이즈만 남음
3. 33개 관측치로 462개 가설을 테스트하면 검정력 0에 수렴

**올바른 태도**: "FDR 유의 0건 = 이 모델에서 투자 결론을 도출하면 안 된다"

---

### 오류 2: 보정 전 p-value를 근거로 투자 가설 구성

초판에서 가장 심각한 오류:

```
"PPI_YOY → Healthcare win5: t=3.22, p=0.003 ★★★"
"실업률 → Consumer Discretionary win21: t=3.04, p=0.005 ★★★"
```

이 값들을 ★★★로 표시하고 투자 근거로 사용했으나:
- FDR 보정 후 p_adj = 0.61 (유의하지 않음)
- 462개 테스트 중 상위 2개가 p<0.01인 것은 **순수 우연(random chance)으로도 기대**됨
- 기대 가양성: 462 × 0.01 = 4.6건 → 실제 3건은 우연 수준 이하

**결론**: 초판에서 ★★★로 표시한 이벤트-섹터 조합은 **통계적 노이즈**에 스토리를 입힌 것.

---

### 오류 3: 확인 편향(Confirmation Bias)에 의한 인과 추론

| 초판 주장 | 실제 |
|----------|------|
| "PPI 상승 → Healthcare 수혜 (가격 전가력)" | PPI 데이터는 합리적이나 p_adj=0.61. 인과가 아닌 우연 |
| "실업률↑ → Cons.Disc↑ (재정부양 기대)" | 역설적 결과에 맞춤형 사후 합리화. 데이터가 그 스토리를 지지하지 않음 |
| "BOK 금리↑ → IT↑ (경제정상화 신호)" | POLICY_RATE 값 자체가 1.5%p 잘못됨. 근거 자체가 무효 |
| "CPI → IT win21 +33.3% (인플레 헤지)" | CPI 값이 2,859~1,230,715. 무의미한 데이터로 계산한 수치 |
| "유틸리티 = 전 레짐 양(+) 유일 섹터" | 레짐 분류 자체의 근거가 불명확 |

**패턴**: 데이터가 직관과 맞으면 → "통계적으로 확인됨" 선언.
데이터가 직관과 안 맞으면(실업률↑ → 소비재↑) → 창의적 스토리로 합리화.
어느 쪽이든 결론이 바뀌지 않는다면, 그것은 분석이 아니라 **사전 신념의 재확인**.

```
초판: "최근 6개월 Win21 평균 → IT +28.2% (가장 높은 이벤트 반응)"

```

```python
    # expected_value NULL 비율 < 20% 검증
```

#### 4b. 확인 편향 방지 프로토콜

```
1. 통계 결과를 먼저 제시 → 스토리는 그 다음
2. FDR 보정 후 유의하지 않은 결과는 "참고" 이상으로 사용 금지
3. 반직관적 결과에 대한 "사후 합리화" 금지 —
   대신 "이 결과는 설명 불가. 추가 데이터 필요" 명시

```

#### 4c. 결론 신뢰도 등급제

| 등급 | 기준 | 사용 가능 범위 |
|:----:|------|-------------|
| **A** | FDR 유의 + 데이터 검증 완료 + 외부 소스 교차 확인 | 투자 의사결정 직접 사용 가능 |
| **B** | 보정 전 p<0.05 + 데이터 정상 + 방향성 일관 | 참고용, 다른 근거와 병행 시 사용 |
| **C** | p>0.05이나 방향성 일관 + 외부 소스 지지 | 탐색적(exploratory) 가설만 |
| **D** | 데이터 손상 또는 모델 전제 위반 | **사용 불가** |

**초판 분석의 대부분은 등급 D.**
